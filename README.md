# Java Swing Calculator

A simple desktop calculator application written in **Java** using **Swing**.

---

## Features

- Basic arithmetic operations:
  - Addition
  - Subtraction
  - Multiplication
  - Division
- Clear / reset functionality
- Intuitive button-based user interface
- Code written with **robustness and error resistance** in mind:
  - Prevents invalid operations (e.g. division by zero)
  - Safely handles incorrect or incomplete input
  - Designed to minimize runtime errors and crashes

---

## How It Works

- The user enters numbers using on-screen buttons
- Operator buttons select the desired arithmetic operation
- Pressing `=` calculates and displays the result
- The interface is built with Java Swing components
- Button actions are handled using event listeners
- Input and calculations are validated to ensure safe execution
